## 📌 Sobre a Disciplina
A disciplina foca nos pilares fundamentais da POO e na aplicação prática utilizando a Java Virtual Machine (JVM). Os principais tópicos abordados incluem:

* **Abstração:** Representação de objetos do mundo real em modelos de código.
* **Encapsulamento:** Proteção de dados e métodos através de modificadores de acesso.
* **Herança:** Reutilização de código e criação de hierarquias entre classes.
* **Polimorfismo:** Capacidade de um objeto ser referenciado de várias formas.
